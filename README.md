# Projeto Zoo 🦁🌿

Projeto desenvolvido com **HTML, CSS, JavaScript e Astro**.

O objetivo do projeto é apresentar diferentes biomas e áreas de um zoológico,
utilizando páginas organizadas e reaproveitamento de componentes com Astro.

## Tecnologias
- HTML
- CSS
- JavaScript
- Astro

## Como rodar o projeto
```bash
npm install
npm run dev
